HTGL Project
Fundamental Distro Builder
1. Technical Debootsraping
   a. Based Debian "Debootsraping"
     - Latest"REturn" Based "Bullseye"
     - Arsip "Revolution Based "Jessie"
   b. Fokus Distro Pentester Live System
     - Architecture amd64 dan i686
     - Size Image ISO Max 2.5 GB
   c. Installation "Optional"
     - HTGL-Clone
      
2. Choice Desktop MATE Environment
   "Inspired by BackTrack Linux R5 Gnome" 
   Mengembalikan kenangan yang hilang dengan gnome2.
   gnome2 sudah tidak ada berganti dengan project MATE fork dari Project GNOME2
   
3. Directory File System Include /pentest "Tools Native"
   "Inspired by BackTrack Linux R5 Gnome" 
   Koleksi Tools Native
   a. Tools Native akan ada script khusus yang akan mengelola versi dari toolsnya.
   b. Tools apt juga ada di https://deb.hacktracklinux.org
   
4. Experience Distro Startup
   "Inspired by BackTrack Linux R5 Gnome" 
   a. Tidak akan ada Login Manager (gdm/lightdm dll)
   b. Login di awal akan selalu TTY untuk gui gunakan startx
