About of this live operating system?

<<Hack|Track GNU/Linux is an open source operating system founder Dindin Hernawan from Indonesia that provides penetration testing. The pentest tool is generated from the source code and then compiles it.

Features of this live operating system?

Simple

Simple does not need to install. Make bootable media usb flash and ready to act.

Modern

Modern look and very easy in using it. There is a dark and light theme.

Open source

Anyone can use, copy, learn, modify, and redistribute.

Free

This free operating system does not have to pay. But if you are interested we accept your donation.

Create bootable devices and ready for action!

Our focus is to create a live system for USB Flashdisk or DVD only. Can not be installed to hard drive or other storage device. Simply create a bootable device with rufus software and perform penetration testing on any computer. Are you ready to act?
