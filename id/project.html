Proyek HTGL

Berikut adalah proyek-proyek yang telah kami kerjakan.

Distro

Kami sedang membangun distribusi HackTrack Linux untuk sistem live penetration testing. Kami menyediakan prototipe atau kandidat rilis. Harap dicatat bahwa versi ini hanya untuk pengujian dan belum stabil. Teknik sistem build yang kami gunakan adalah debootstrapping. Sumber-sumber alat pentest tersedia di organisasi GitHub kami.

Panduan

Kami sedang membangun dokumentasi HackTrack.
1. Panduan untuk membuat Bootable di HackTrack Linux.
2. Panduan Desktop Standar Linux dan Menu HackTrack.
3. Panduan alat penetrasi Linux Hacktrack.
